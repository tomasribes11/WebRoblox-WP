# Configurar Menú de Categorías WordPress

Plan para configurar el menú superior que funcione correctamente con categorías, mostrando solo artículos de cada sección al hacer clic.

## Análisis Actual
- Categorías requeridas ya existen: comparativas (4 posts), guias (4 posts), errores (4 posts), calculos (7 posts), tutoriales (1 post)
- Múltiples menús disponibles, usaremos "Menu Categorías Funcional" para empezar limpio
- 15 posts totales, muchos necesitan asignación correcta de categorías

## Pasos a Ejecutar

### 1. Asignar Categorías a Posts
- Revisar cada post y asignar categoría correcta
- Eliminar posts de "Uncategorized"
- Asegurar que cada post tenga SOLO una categoría

### 2. Crear Menú Funcional
- Usar "Menu Categorías Funcional" (actualmente vacío)
- Añadir las 5 categorías como elementos del menú
- Configurar orden lógico

### 3. Asignar Menú a Ubicación
- Asignar el menú a "Primary" o "Header"
- Reemplazar menú actual si es necesario

### 4. Validación Final
- Probar cada enlace del menú
- Verificar que muestre solo posts de esa categoría
- Confirmar que no haya mezcla de contenido

## Resultado Esperado
Menú completamente funcional donde cada opción lleva a su categoría correspondiente mostrando únicamente los artículos de esa sección.
